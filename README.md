# Analyse des données d’activité – Bellabeat

##  Contexte du projet

Bellabeat est une entreprise spécialisée dans les objets connectés dédiés à la santé des femmes.  
L’objectif de ce projet est d’analyser les données d’activité issues de dispositifs connectés afin de comprendre le comportement des utilisatrices et identifier des opportunités d’amélioration de l’engagement produit.

---

## Objectifs de l’analyse

- Comprendre le niveau d’activité moyen des utilisatrices
- Identifier la proportion de journées sédentaires, modérées et actives
- Analyser la relation entre activité physique et calories brûlées
- Mettre en évidence des indicateurs clés de performance (KPIs)
- Proposer des recommandations business basées sur les données

---

## Données utilisées

Le jeu de données contient des informations journalières sur :

- Nombre total de pas
- Calories brûlées
- Minutes d’activité intense
- Minutes sédentaires
- Identifiant utilisateur
- Date d’activité

Les données ont été nettoyées et préparées via SQL (MySQL).

---

## Outils et technologies

- MySQL : nettoyage et analyse des données
- SQL : création des KPIs et des vues analytiques
- GitHub : documentation et structuration du projet
- Tableau : pour la visualisation

---

##  KPIs principaux

- Moyenne des pas quotidiens
- Moyenne des calories brûlées
- Répartition des niveaux d’activité (Sédentaire,Modéré,Actif)
- Indicateurs d’engagement utilisateur

---

##  Méthodologie

1. Importation des données CSV dans MySQL
2. Vérification de la qualité des données (doublons, valeurs manquantes)
3. Création d’indicateurs métier via SQL
4. Création de vues propres pour la Business Intelligence
5. Interprétation des résultats et recommandations
